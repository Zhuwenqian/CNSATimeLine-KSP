# 功能更新

## 文件用途

本文档用于按时间顺序记录 CNSATimeLine 项目的功能新增、Bug 修复和重大变更。每条记录应包含日期、变更类型和简要说明。当文档超过 200 行时，应删除最早的几条记录以保持可读性。

---

### 2026-07-04：太空中心时间线 UI 与自动加速

- **变更类型**：新增
- **变更内容**：
  - 将 `CNSATimeLineMod.cs` 从飞行场景入口改为太空中心（SpaceCentre）场景入口。
  - 新增 `TimeLineEvent.cs`，定义时间线事件数据结构。
  - 新增 `TimeLineConverter.cs`，通过反射读取 `RSSTimeFormatter` 的 epoch，实现真实日期时间与 KSP UT 秒数的映射。
  - 新增 `TimeLineDataLoader.cs`，解析 `中国航天大事时间线.txt` 为事件列表。
  - 新增 `TimeLineWindow.cs`，实现太空中心场景的 IMGUI 窗口，支持 `Ctrl+L` 打开/关闭。
  - 每个事件旁显示“加速到此时”按钮，点击后通过 `TimeWarp.fetch.WarpTo` 加速到事件时间前 10 小时；不可倒退加速的按钮自动置灰。
  - 更新 `CNSATimeLine.csproj` 以包含新增源文件。
  - 更新 `readme/开发文档.md`、`readme/实施文档.md` 与 `readme/功能更新.md`。

---

### 2026-07-04：项目初始化

- **变更类型**：新增
- **变更内容**：
  - 初始化 KSP 1.12.5 模组开发环境。
  - 创建 `CNSATimeLine.csproj`，目标框架为 .NET Framework 4.7.2。
  - 配置对 `KSP DLL/` 目录下 KSP 与 Unity 程序集的引用。
  - 配置对 `RSSTimeFormatter.dll` 的引用，并设置 `Private=false` 避免构建输出冗余。
  - 创建 `CNSATimeLineMod.cs` 基础模组入口类示例，演示 RSSTimeFormatter 依赖检测。
  - 创建 `readme/` 目录及开发文档、实施文档、功能更新文档。
